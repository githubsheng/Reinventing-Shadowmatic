about
Nowadays browsers are capable of displaying complex 3D objects. When a web designer finish making the model, they often want to see how it looks like in a real browser. This tiny application allows you to upload a .obj file produced by your favourite modeling tools, and it will try to render the objects. This application uses a library called glMatrix for fast matrix calculations.
- Drag and rotate the node on the right. Light and shadows calculated real time.
- You can upload any .obj files and this library will render it.
- This library uses glMatrix for fast matrix calculation
